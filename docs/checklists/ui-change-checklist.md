# UI Change Checklist

- Test authenticated app in light mode.
- Test authenticated app in dark mode.
- Test system mode.
- Test splash page while OS/app is in dark mode.
- Confirm splash remains light-only.
- Test mobile width.
- Test desktop width.
- Check hover/focus states.
- Check form validation states.
- Check empty states.
- Check loading/pending states.
- Check contrast for muted text.
- Check metric chips and badges.
- Check dialogs/popovers.
- Check no horizontal overflow.
