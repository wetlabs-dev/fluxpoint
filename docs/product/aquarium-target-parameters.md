# Aquarium target parameters

The aquarium target profile stores temperature, pH, GH, KH, nitrogen thresholds, and salinity. Structured minimum and maximum values drive metric threshold lines; target values provide a useful midpoint for temperature, pH, GH, and KH.

Eddy Parameter Advisor can propose ranges based on the intersection of active stocked species with complete saved data. It preserves missing data as uncertainty and reports non-overlapping stocking as a conflict rather than inventing a compromise. Advice remains review-only until a Collection Owner or Aquarist explicitly applies selected safe changes.

Ammonia and nitrite always remain 0 ppm. Nitrate is an upper threshold, not a goal. Stable water chemistry is safer than chasing exact pH or hardness values, so sensitive adjustments should be gradual and verified against trusted husbandry sources.
