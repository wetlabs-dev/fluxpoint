# Species aliases

Species aliases are collection-scoped alternate names attached to a species definition. They support common names, trade names, old names, misspellings, scientific synonyms, local names, and uncategorized alternatives.

Aliases can be added manually or reviewed from a Species Magic Fill draft. They are normalized for case-insensitive duplicate prevention while retaining the keeper's display spelling. Notes and source attribution are optional.

The Species library includes aliases in search and displays them on library cards and species detail pages. Because aliases belong to the current collection, a collection can add its own terminology to a shared starter definition without changing the shared canonical record.
