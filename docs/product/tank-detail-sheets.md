# Aquarium livestock detail sheets

From a tank’s Settings → QR / Labels section, a Fishkeeper can generate a letter-size livestock detail sheet. It lists current fish, invertebrates, plants, and other living inventory, grouped by category with quantity, status, scientific or common name, aliases where available, and a stable QR code for each item.

Archived, consumed, dead, removed, and transferred items are omitted. Long lists paginate automatically and repeat the tank header. Empty tanks produce a valid sheet with an explicit empty-state message. The generated PDF is stored and protected by the same collection access rules as individual labels.
